## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/favrora/food-order-app.git
   ```
2. From the root, install NPM packages
   ```sh
   npm install
   ```
3. Start the vite app:
   ```sh
   npm run dev
   ```
